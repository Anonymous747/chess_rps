# Profile Avatars

This directory should contain 20 avatar images named:
- avatar_1.png
- avatar_2.png
- avatar_3.png
- ... (up to avatar_20.png)

Each avatar should be:
- Square format (recommended: 256x256 or 512x512 pixels)
- PNG format with transparency support
- Unique design representing different chess-themed characters

The avatars will be displayed in:
- Profile screen (user's current avatar)
- Collection screen (avatars tab)
- Leaderboard (if implemented)
